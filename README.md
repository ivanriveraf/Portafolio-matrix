# Portafolio Estilo Matrix

Este es la maqueta de mi portafolio personal inspirado en el estilo de *Matrix*, hecho con HTML, CSS, Bootstrap, JavaScript y jQuery.

## Características actuales
- Fondo animado estilo Matrix
- Animaciones al hacer scroll
- Sección emergente al hacer clic en la imagen de perfil
- Responsive con Bootstrap
- Código organizado y listo para GitHub

## Vista previa
Puedes abrir `index.html` en cualquier navegador para ver el sitio.
